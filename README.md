#  Expense Tracker

A simple and interactive **Expense Tracker** built using HTML, CSS, and JavaScript. This application helps users manage their finances by tracking income and expenses in real time.
##  Features

*  Add income and expense transactions
*  Automatic balance calculation
*  Separate tracking of income and expenses
*  Transaction history list
*  Option to remove transactions (if implemented in JS)
*  Clean and responsive user interface

##  Technologies Used

* HTML
* CSS
* JavaScript

##  How It Works

* Enter a **description** of the transaction
* Enter the **amount**

  * Use **positive values (+)** for income
  * Use **negative values (-)** for expenses
* Click **Add Transaction**
* The balance, income, and expense values update automatically
* Transactions appear in the list
